Naïve Bayes Algorithm from Scratch

Implement the Naïve Bayes algorithm from scratch without using any external libraries like sklearn. The algorithm should work for text classification, particularly for spam detection.

Instructions:

  • Implement functions to preprocess text, calculate probabilities, and classify new messages.
  • Do not use any machine learning libraries.
  • Follow a non-class-based approach (no OOP).
  • Include test cases to validate the implementation.

Example:

Given the dataset:
[("free money", "spam"), ("hello friend", "ham"), ("win lottery now", "spam"), ("meeting at 5", "ham")]
The model should correctly classify: "win cash now" as spam.
