Random Forest Algorithm

Random Forest is an ensemble learning method that builds multiple decision trees and aggregates their predictions.

Mathematical Formulation

For classification, the final prediction is obtained using majority voting:

\( \hat{y} = \text{mode} \{ T_1(x), T_2(x), ..., T_M(x) \} \)

For regression, the final prediction is the average of all tree outputs:

\( \hat{y} = \frac{1}{M} \sum_{i=1}^{M} T_i(x) \)

Implementation Steps

  1. Create multiple bootstrap samples from the dataset.
  2. Train a decision tree on each bootstrap sample.
  3. For each split, choose a random subset of features.
  4. Aggregate predictions:
    • Classification: Use majority voting.
    • Regression: Take the average prediction.

Example Calculation

Given a dataset with multiple features, we train multiple decision trees:

  • Suppose we train 5 decision trees on different bootstrapped samples.
  • Each tree makes a prediction.
  • For classification, we select the most common prediction.
  • For regression, we take the average of all predictions.

Final Prediction

After training multiple trees and aggregating predictions:

  • If performing classification, we use majority voting.
  • If performing regression, we average the tree predictions.
