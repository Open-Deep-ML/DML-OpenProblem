Generate Confusion Matrix

Write a Python function that generates the confusion matrix for a list of results of a binary classification task

Each observation has two values: the actual label and the predicted label

Example:

  • Input: [[1, 1], [1, 0], [0, 1], [0, 0], [0, 1]]
  • Output: [[1, 1], [2, 1]]
  • Reasoning:
    • There is one observation of class 1 that was labeled as class 1 ([1,1]), so there is 1 true positive
    • There is one observation of class 1 that was labeled as class 0 ([1,1]), so there is 1 false negative
    • There are two observations of class 0 that were labeled as class 1 ([0,1]), so there are 2 false positives
    • There is one observation of class 0 that was labeled as class 0 ([0,0]), so there is 1 true negative
