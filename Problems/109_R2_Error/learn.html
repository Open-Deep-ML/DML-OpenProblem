R-squared (\( R^2 \)) Calculation

R-squared (\( R^2 \)) is a statistical measure that represents the proportion of the variance in the dependent variable that is explained by the independent variables in a regression model.

Mathematical Formulation

R-squared is defined as:

\[ R^2 = 1 - \frac{\sum_{i=1}^{n} (y_i - \hat{y}_i)^2}{\sum_{i=1}^{n} (y_i - \bar{y})^2} \]

  • \( n \): Total number of data points
  • \( y_i \): Actual value at index \( i \)
  • \( \hat{y}_i \): Predicted value at index \( i \)
  • \( \bar{y} \): Mean of actual values
  • \( \sum (y_i - \hat{y}_i)^2 \): Sum of squared errors (SSE)
  • \( \sum (y_i - \bar{y})^2 \): Total sum of squares (TSS)

Implementation Steps

  1. Compute the mean of actual values: \[ \bar{y} = \frac{1}{n} \sum_{i=1}^{n} y_i \]
  2. Calculate the Sum of Squared Errors (SSE): \[ SSE = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 \]
  3. Calculate the Total Sum of Squares (TSS): \[ TSS = \sum_{i=1}^{n} (y_i - \bar{y})^2 \]
  4. Compute \( R^2 \) using: \[ R^2 = 1 - \frac{SSE}{TSS} \]

Example Calculation

Given the actual and predicted values:

  • Actual values: \( y = [3, -0.5, 2, 7] \)
  • Predicted values: \( \hat{y} = [2.5, 0.0, 2, 8] \)

Step-by-step calculation:

  • Compute \( \bar{y} \): \[ \bar{y} = \frac{3 + (-0.5) + 2 + 7}{4} = 2.875 \]
  • Compute SSE: \[ (3-2.5)^2 + (-0.5-0.0)^2 + (2-2)^2 + (7-8)^2 = 0.25 + 0.25 + 0 + 1 = 1.5 \]
  • Compute TSS: \[ (3-2.875)^2 + (-0.5-2.875)^2 + (2-2.875)^2 + (7-2.875)^2 = 0.0156 + 11.3906 + 0.7656 + 17.0156 = 29.1875 \]
  • Compute \( R^2 \): \[ R^2 = 1 - \frac{1.5}{29.1875} = 0.9486 \]

The R-squared value for this example is 0.9486, indicating a strong fit of the model.
